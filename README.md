# RICE NAVI 炊飯トラブル解決アプリ v1.2

更新ファイル `炊飯ナレッジDB_統合マスター_v12.5_Excel版_PC用(1).xlsx` を元に作成したスマホ向け軽量Webアプリです。

## 内容

- アプリ名：RICE NAVI
- 用途：炊飯トラブル解決
- 診断件数：31件
- 知識DB：839件（全件格納）
- 診断直結ナレッジ：97件
- 出典DB：47件（全件格納）
- 診断直結出典：22件
- 形式：静的HTML / CSS / JavaScript
- PWA対応：スマホにインストール可能
- オフライン対応：初回表示後に主要ファイルをキャッシュ

## v1.2での修正

v1.1では、知識DB全体ではなく「トラブル診断に直接ひもづく関連ナレッジ」だけを格納していたため、97件表示になっていました。
v1.2では、Excelの知識DB 839件を全件格納し、アプリ内の「ナレッジ検索」から検索できるようにしています。

画面を軽くするため、ナレッジ検索は全件を一度に画面表示せず、先頭120件まで表示します。
検索語を入力すると絞り込み表示できます。

## 使い方

1. ZIPを解凍します。
2. 中のファイルをGitHub Pagesのリポジトリにアップロードします。
3. `index.html` を開くとアプリが表示されます。
4. スマホではブラウザメニューからホーム画面に追加できます。
