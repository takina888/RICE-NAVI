# RICE NAVI v1 - 炊飯トラブル解決アプリ

この版は、Excelの「トラブル診断」シートを中心にしたスマホ向け軽量PWAです。
前回のような「知識カードを読むアプリ」ではなく、現場トラブル時に使う構成です。

## 主な機能

- 症状検索
- よく使う症状チップ
- 分類フィルター
- 緊急度フィルター
- 最重要トラブルだけ表示
- 診断詳細：原因候補、確認項目、試験例、やってはいけないこと
- 関連ナレッジ表示
- 出典表示
- 保存機能
- 現場メモ（端末内保存）
- PWA対応、GitHub Pages対応

## データ

- トラブル診断：40件
- 関連ナレッジ：118件
- 出典：28件

## 使い方

ZIPを解凍し、フォルダ内のファイルをGitHub Pagesのリポジトリにアップロードしてください。
`index.html` が入口です。
