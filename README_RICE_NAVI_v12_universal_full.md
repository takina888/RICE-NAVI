# RICE NAVI v12 universal full

RICE NAVIの日本語版UXを成立させるためのv12です。

## 重点修正

- 今日の米占いをホーム最上部に配置
- RNアイコンを正式アイコンとして使用
- ユニバーサルデザイン方針を反映
- 戻れない画面をなくすため、全詳細画面に「← ホームへ」導線を設置
- ホーム上部コンテンツは開閉式中心
- 「例：日本」「面白さから入る」などの開発メモ表示を削除
- 炊飯文献ライブラリはカテゴリー→タイトル一覧→詳細表示に変更
- 文献詳細は横スクロール禁止、長いIDも折り返し
- お米マイスター100は初級30・中級40・上級30の学習コースとして再構成
- 米品種図鑑は空画面にせず、未確認数値は未確認と表示
- お米の未来はタイトル一覧→本文表示
- 水、環境、納米庫のチェック機能を入力→判定形式で実装

## 方針

まず日本語版で「戻れる、読める、学べる、空画面がない」を満たすことを優先しています。
多言語はこのUXが固まった後に再適用します。
