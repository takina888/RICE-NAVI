# RICE NAVI v41 UI Grade-up

## 目的
v40をベースに、初めて使うユーザーが迷いにくく、目で見て楽しめるUIへ寄せた版です。

## 主な変更
- ホームに「初めての方へ」「今日見る3つ」導線を追加
- 米占い、米天気、世界ライス物語をすぐ分かる入口として整理
- アイコン・挿絵風のCSS装飾を追加
- 文献ページの「エビデンスA/B/C/D」見えを廃止
- 文献の優劣ではなく「根拠確認ステータス」として表示
- 原文確認済み候補 / 要約整理済み / 確認中 / 現場メモ に表記変更
- GPS・天気・水質は測定ではなく補助情報である説明を維持
- data/rice_navi_data_v41.json を読み込み
