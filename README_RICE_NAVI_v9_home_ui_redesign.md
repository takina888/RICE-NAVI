# RICE NAVI v9 home UI redesign

## 目的
RICE NAVIを「硬いDB表示」ではなく、毎日開きたくなるナビ・発見・学習アプリの見た目に寄せた版です。

## 主な変更
- 「今日の米占い」をホーム最上部に配置。
- ホームメニュー順を整理。
- 学習モジュール名を「お米マイスター100」に統一。
- 炊飯文献ライブラリを信頼性の中心として強調表示。
- お米マイスター100内に「温度と時間の関係」表を配置。
- 下部ナビを追加：ホーム / 学ぶ / 文献 / チェック / 状況。
- 旧RICE WORKS系の学習カードはRICE NAVI内の文献起点学習DBとして表示。
