このZIPは GitHub Pages へアップロードする公開用ファイルのみを含みます。

## m70の位置づけ

- m69を基準にしたホーム導線・公開表示の再監査版です。
- 新規の大規模翻訳追加ではなく、ホームが入口として機能するか、表示文が言語切替時に不自然になりにくいかを確認しました。
- 文献カード翻訳は途中で止めています。
- RWLF0001〜RWLF0240 は多言語タイトル/要約フィールド追加済みです。
- RWLF0241〜RWLF0428 は翻訳確認中として扱います。
- 4言語対応は進行中で、完全完了とは扱いません。
- 用語集は1000語です。
- 公開用ZIPにはExcel、作業メモ、監査用中間資料を入れません。

## アップロード手順

ZIP内のファイルをリポジトリ直下へアップロードしてください。
`index.html`, `app.js`, `styles.css`, `sw.js`, `manifest.json`, `version.txt`, `data/`, `assets/` を同じ階層に置きます。
