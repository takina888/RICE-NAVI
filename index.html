このアプリについて

RICE NAVIは、炊飯ナレッジDBをスマホで検索しやすくした軽量Webアプリです。データは端末内で表示され、サーバー処理や外部AI処理は使っていません。

運用方法

GitHub Pagesにこのフォルダをそのままアップロードすると公開できます。Excelを更新した場合は、data.jsを作り直す方式です。

初回表示後は、対応ブラウザではオフライン表示用にファイルをキャッシュします。
